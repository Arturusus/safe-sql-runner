# Безопасный SQL Runner

## Описание
Консольный скрипт на Python для безопасного выполнения SQL-запросов к PostgreSQL. 
Программа пропускает только SELECT-запросы, автоматически добавляет LIMIT 5 и блокирует запрещенные команды.

## Запуск
1. Установите зависимости:
pip install -r requirements.txt

2. Создайте файл .env на основе .env.example и укажите в нем строку подключения DATABASE_URL.

3. Запустите скрипт:
python main.py
